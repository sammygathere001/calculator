# JavaScript Calculator

This project is a simple calculator built with **HTML**, **CSS**, and **JavaScript**.

I made it as one of my first learning projects while practicing JavaScript and getting more comfortable with how JavaScript interacts with the DOM.

## About The Project

The calculator lets users:

- Enter numbers
- Use basic operators like addition, subtraction, multiplication, and division
- Add decimals
- Clear the display
- Calculate the result

This project helped me practice:

- DOM selection
- Event listeners
- Using `data-*` attributes
- Updating the UI with JavaScript
- Writing logic for calculator operations

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Getting Started

To run this project locally:

1. Clone or download the repository
2. Open the project folder
3. Open `index.html` in your browser
